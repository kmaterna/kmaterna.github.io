Welcome to Tectonic_Utils’s documentation¶

Welcome to a Tectonic_Utils, a simple library with utility functions for “everyday use” in active tectonics research. There are geodesy and seismology utility functions in Python, some originally from libraries in Matlab or other languages. There are read-write functions to access commonly used formats. Feel free to explore and use!

Disclaimer: Completely bug-free code can never be guaranteed. As things come up, I occasionally fix and update the package on Pip/PyPI. If you’re using this package and you’d like to contribute or you find bugs, please let me know.
